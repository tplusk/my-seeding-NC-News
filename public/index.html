NC News

A full-stack news application that allows users to read articles, comment, and interact with content.

List of available endpoints:

  • GET /api
  • GET /api/topics
  • GET /api/articles
  • GET /api/articles/:article_id
  • GET /api/articles/:article_id/comments
  • GET /api/users
  • POST /api/articles/:article_id/comments
  • PATCH /api/articles/:article_id
  • PATCH /api/comments/:comment_id
  • DELETE /api/comments/:comment_id

GET /api

Returns API documentation overview.

Example Response

{
  "endpoints": [
    { "method": "GET", "path": "/api", "description": "API documentation" }
  ]
}

GET /api/topics

Retrieves a list of all available topics.

Example Response

{
  "topics": [
    { "slug": "coding", "description": "Code-related news" }
  ]
}

GET /api/articles

Retrieves all articles. Supports filtering & sorting via query parameters.

Query Parameters

Parameter Type Description
topic string Filter articles by topic
sort_by string Field to sort by (e.g., created_at)
order string Sort order (asc/desc)

Example Response

{
  "articles": [
    { "article_id": 1, "title": "Intro to APIs", "votes": 10 }
  ]
}

GET /api/articles/:article_id

Retrieves a specific article by its ID.

Example Response

{
  "article": { "article_id": 1, "title": "Intro to APIs", "votes": 10 }
}

GET /api/articles/:article_id/comments

Retrieves all comments for a given article.

Example Response

{
  "comments": [
    { "comment_id": 1, "body": "Great article!", "votes": 3 }
  ]
}

GET /api/users

Retrieves all users.

Example Response

{
  "users": [
    { "username": "kevingarnett", "name": "Kevin Garnett" }
  ]
}

POST /api/articles/:article_id/comments

Posts a new comment to a specific article.

Request Body

{
  "username": "jerryrice",
  "body": "This article is awesome!"
}

Example Response

{
  "comment": {
    "comment_id": 2,
    "body": "This article is awesome!",
    "votes": 0
  }
}

PATCH /api/articles/:article_id

Updates the vote count for a specific article.

Request Body

{ "inc_votes": 1 }

Example Response

{
  "article": { "article_id": 1, "votes": 11 }
}

PATCH /api/comments/:comment_id

Updates the vote count for a specific comment.

Request Body

{ "inc_votes": -1 }

Example Response

{
  "comment": { "comment_id": 1, "votes": 2 }
}

DELETE /api/comments/:comment_id

Deletes a specific comment. Returns no content.
